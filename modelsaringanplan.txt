# Rencana Pengembangan Model EfficientNetV2B2 untuk PWA PaddyPadi

## üéØ Tujuan Utama
- Mengembangkan model deteksi penyakit padi yang ringan dan cepat
- Mengoptimalkan untuk inferensi real-time di PWA
- Mengkonversi ke format TensorFlow.js

## üìã Fase Pengembangan

### 1. Persiapan Dataset
- Menggunakan dataset penyakit padi dari Kaggle
- Pembagian data:
  - Training: 80%
  - Validasi: 20%
- Preprocessing gambar:
  - Resize ke 256x256 (ukuran standar EfficientNetV2B2)
  - Normalisasi piksel

### 2. Data Augmentasi
- Teknik augmentasi:
  - Random flip (horizontal/vertical)
  - Random rotation (¬±20¬∞)
  - Random zoom (¬±15%)
  - Random brightness/contrast
  - Random crop
  - dan beberapa augmentasi lain

### 3. Arsitektur Model
- Base model: EfficientNetV2B2 (pre-trained ImageNet)
- Modifikasi arsitektur:
  - Remove classification head
  - Add GlobalAveragePooling2D
  - Add Dropout(0.2)
  - Add Dense layer (10 classes, softmax)

### 4. Training Strategy
- Initial training:
  - Freeze base model layers
  - Train head layers (15-20 epochs)
- Fine-tuning:
  - Unfreeze last 15-20 layers
  - Very small learning rate (1e-5)
  - Train additional 10-15 epochs

### 5. Model Optimization
- Evaluasi performa:
  - Target accuracy: >75%
  - Monitor inference speed
- Quantization:
  - Convert to uint8
  - Reduce model size
  - Optimize for mobile devices

### 6. Konversi ke TensorFlow.js
- Menggunakan tensorflowjs_converter
- Optimasi:
  - Weight quantization
  - Layer optimization
  - Generate web-friendly format

### 7. Deployment Preparation
- Format output: .json model topology
- Sharded weight files
- Metadata configuration
- Browser compatibility check
